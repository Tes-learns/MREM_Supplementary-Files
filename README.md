This study investigates the use of a finite mixture model for the random-effects distribution as a flexible alternative for modelling between-study heterogeneity. 
Parameter estimation is performed using a likelihood-based Expectation–Maximisation (EM) algorithm. 
A simulation study was conducted under unimodal and multimodal random-effects distributions, varying the number of studies and the number of mixture components. 
Model performance was evaluated using the root mean square error (RMSE), the Akaike Information Criterion (AIC), and the Bayesian Information Criterion (BIC).
